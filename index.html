HTML/CSS

HTML consists of a series of elements, which you use to enclose, or wrap, different parts of the content to make it appear a certain way, or act a certain way. The enclosing tags can make a word or image hyperlink to somewhere else, can italicize words, can make the font bigger or smaller, and so on.

CSS describes how elements should be rendered on screen, on paper, in speech, or on other media.

JavaScript

Designed by: Brendan Eich of Netscape initially; others have also contributed to the ECMAScript standardJavaScript runs on the client side of the web, which can be used to design / program how the web pages behave on the occurrence of an event.

ReactJs

React is a library for building composable user interfaces. It encourages the creation of reusable UI components, which present data that changes over time. Lots of people use React as the V in MVC. React abstracts away the DOM from you, offering a simpler programming model and better performance. React can also render on the server using Node, and it can power native apps using React Native.
